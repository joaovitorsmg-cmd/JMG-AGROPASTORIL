API Key e preferências do painel

🔑 API Key da Anthropic

Necessária apenas para o Chat Consultor. Todas as calculadoras, clientes, aprendizado e registro de visita funcionam sem ela.

Obtenha em console.anthropic.com → API Keys → Create Key.

Salva só neste dispositivo. Nunca sai daqui.

💾 Backup dos dados

Clientes, compras, visitas e progresso ficam salvos só neste aparelho. Exporte um backup de vez em quando — se trocar de celular ou limpar o navegador, você importa de volta.

☁️ Backup automático na nuvem

Sem isso, se você perder o celular, trocar de aparelho ou ele for roubado, os dados se perdem para sempre. Conectando, toda alteração é enviada automaticamente para sua conta gratuita do JSONBin.io — em qualquer aparelho novo, basta colar a key e o ID do backup para restaurar tudo.

O plano gratuito do JSONBin é mais que suficiente — o backup inteiro deste painel tem só alguns KB. Guarde a Master Key em local seguro: com ela é possível ler e alterar o backup.
